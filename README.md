# Tiny Stocks

轻量 A 股自选看板与 Windows 悬浮窗。

## 网页版

打开 `stock-board.html`，支持：

- 名称、代码、拼音实时搜索候选
- 自定义刷新间隔，最低 1 秒
- 顶部指数胶囊与紧凑自选列表
- 点击股票/指数进入二级详情
- 当日、5 日分时、日 K、周 K、月 K、年 K
- 深色、浅色、跟随系统三种主题
- 无账号云端同步：用同步码保存自选、刷新间隔和主题
- 图表悬浮读数、K 线拖拽平移、滚轮缩放
- 自选行拖拽排序
- 点击右侧指标可切换为主显示项

## 悬浮窗

双击 `run_stock_float.bat` 启动。新版为无边框角落悬浮窗，支持：

- 实时行情 + 分时火花线
- 右键添加、删除、刷新、切换置顶
- 红黄绿自绘窗口控件
- 透明度滑块
- 拖动股票行排序
- 点击右侧指标切换主显示项
